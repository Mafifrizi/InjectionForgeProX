# Security Policy

InjectionForge Pro X is a security testing tool. Use it only against systems where you have explicit authorization.

## Reporting issues

For project security issues, open a private advisory or contact the maintainer through the repository owner profile.

Do not include real credentials, customer data, production tokens, or private target evidence in public issues.

## Data handling

Reports are redacted by default. Runtime artifacts are excluded from version control through `.gitignore`.
